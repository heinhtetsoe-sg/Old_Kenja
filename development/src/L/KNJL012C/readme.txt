KNJL012C
志願者数確定処理(受付確定処理)
-----
KNJL012H を元に作成
-----
学校：智辯
-----

2008/11/13  1.新規。初版。
            2.入試区分関連の修正。

2009/01/17  1.後期入試の場合、前期入試の合格者を除く。

2013/01/21  1.カレッジ入試の場合、合格者は除かない。

2017/12/14  1.五條の後期入試の場合、前期又は自己推薦の合格者を除く。

2018/12/07  1.五條の場合、前期専願Ⅱおよび前期併願でAB合格したものは、後期受験可

2018/12/11  1.和歌山中学の場合、前期G合格者で入学手続済は、後期受験可

2019/01/10  1.カレッジ中学B日程の場合、特色やA日程の合格者を除く。但し、第2志望合格者は、B日程受験可
            -- 第2志望合格者とは、専S選抜・総合や併S選抜・総合で総合に合格したもの

2019/01/11  1.和歌山中学の場合、前期G合格者で入学手続済み無しに係わらず、後期受験可
